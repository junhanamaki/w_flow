## Dependencies

Tested with:

  * ruby 2.2.0, 2.1.1, 2.0.0

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'w_flow'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install w_flow
